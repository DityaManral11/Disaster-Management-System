# 🌍 Disaster Management System

## 📌 Problem Statement

During natural disasters and emergencies, citizens often struggle to quickly reach rescue teams, shelters, and volunteers. Existing communication systems can be slow, fragmented, and difficult to coordinate, leading to delays in emergency response.

---

## 💡 Solution

The Disaster Management System is a centralized platform that connects citizens, volunteers, shelters, and administrators to improve emergency response and disaster coordination. The platform enables SOS reporting, volunteer task management, shelter access, and real-time monitoring through a unified dashboard.

---

## 🚀 Key Features

### 👤 User Module

* User Registration & Login
* Secure Authentication
* Profile Management
* Emergency SOS Requests
* View Disaster Alerts
* Find Nearby Shelters

### 🚨 SOS Management

* Create SOS Requests
* Live Location Support
* Emergency Type Selection
* Real-Time Status Tracking
* Admin Response Management

### 🤝 Volunteer Module

* Volunteer Registration
* Skill & Availability Management
* Task Assignment by Admin
* Volunteer Dashboard
* Assigned Task Tracking

### 🏠 Shelter Management

* Shelter Listings
* Capacity Information
* Available Space Tracking
* Contact Details
* Google Maps Directions

### 🛡️ Admin Module

* Monitor SOS Requests
* Update SOS Status
* Manage Volunteers
* Assign Volunteer Tasks
* Dashboard Analytics
* Resource Monitoring
