# Disaster Management System

A full-stack web application for disaster response and emergency management.

## Features

- User Registration and Login
- SOS Request System
- Emergency Alerts
- Shelter Information
- Volunteer Registration
- Volunteer Task Assignment
- Admin Dashboard
- Profile Management

## Tech Stack

Frontend:
- React.js
- Vite
- Tailwind CSS
- Axios

Backend:
- Node.js
- Express.js
- MySQL

## Environment Variables

```env
DB_HOST=your_aiven_host
DB_PORT=11265
DB_USER=avnadmin
